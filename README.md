# mtools

新しい環境で、自分のお気に入りツール群を一発でセットアップするための個人用インストールスクリプト集です。

## 動作環境

- Windows (WSL) — Ubuntu / Debian 系を想定
- シェル: bash

## 前提条件

以下がインストール済みであること。

- `curl`
- `git`

以下の権限があること。

- `sudo` 権限

## インストール

以下のコマンドを実行するだけで、すべてのツールがインストールされます。

```bash
curl -fsSL https://raw.githubusercontent.com/maechiu/mtools/main/install.sh | bash
```

インストール完了後、シェルの設定を反映させます。

```bash
source ~/.bashrc
```

## アンインストール

対話型メニューから削除したいツールを複数選択してアンインストールできます。

```bash
bash uninstall.sh
```

番号を入力して選択/解除、`Enter` で実行します。`a` で全選択、`n` で全解除、`q` でキャンセルします。

## インストール後の状態

`~/.bashrc` に以下のフックが自動追加されます。

```bash
eval "$(starship init bash)"
eval "$(direnv hook bash)"
export BUN_INSTALL="$HOME/.bun"
export PATH="$BUN_INSTALL/bin:$PATH"
export PATH="$HOME/.anyenv/bin:$PATH"
eval "$(anyenv init -)"
```
